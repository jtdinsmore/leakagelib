`leakagelib_bkg` API documentation
==================================

This code flags IXPE tracks as background particle or photon, assigning each a "particle character". The particle character is related to the probability of the particle representing a particle or photon. Please see :doc:`background_estimation` for instructions on the command line interface for this code. This documentation describes the python API.
